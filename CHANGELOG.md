# Changelog

所有重要更改都将在本文件中记录。

## [1.0.0] - 2024-03-22

### 🎉 首次发布

#### 新增
- 🎧 **实时助听引擎**
  - 毫秒级延迟的音频处理管道
  - 自适应噪音抑制和人声增强
  - 支持耳机和骨传导设备

- 👂 **智能听力测试**
  - 纯音测听 (250Hz-8kHz)
  - Hughson-Westlake 简化测试方法
  - 自动听力图生成

- 🎯 **个性化音频处理**
  - 基于听力图的自动处方计算
  - 7段 EQ 参数自适应调整
  - 用户反馈实时修正

- 🗣️ **高级语音增强**
  - 多频点自适应提升 (200Hz~4kHz)
  - 自动啸叫检测和抑制
  - 骨传导设备专项优化

- 💳 **灵活付费系统**
  - 终身/年付/月付/日卡多种方案
  - Stripe 支付集成
  - 离线模式支持

- 🌍 **国际化**
  - 中文完整本地化
  - 英文界面支持
  - 自动系统语言检测

- 🎨 **现代化UI/UX**
  - 科技感的设计风格
  - 高级滑块和控制组件
  - 实时频谱可视化

#### ⚙️ 技术基础
- React Native 0.81.5 + Expo 54
- TypeScript 5.9 严格模式
- Zustand 状态管理
- Supabase 后端集成
- Next.js 管理后台

#### 📚 文档
- 完整的项目架构文档
- 详细的开发指南
- 音频处理技术细节
- API 文档

#### 🧪 质量保证
- Jest 单元测试
- 音频回归测试
- 70%+ 代码覆盖率
- TypeScript 类型安全

## 快速升级指南

### 从某个版本升级到最新

```bash
# iOS
npm run ios:build

# Android
npm run android:build

# 或使用 EAS
eas build --platform ios
eas build --platform android
```
