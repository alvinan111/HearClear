# 🔧 HearClear 项目改造指南\n\n## 概览\n\n本文档总结了对 HearClear 项目的全面改造，涵盖错误处理、日志系统、安全存储、测试框架、性能监控等方面。\n\n---\n\n## 1️⃣ 错误处理与日志\n\n### 新增文件\n- `src/services/logger.ts` - 统一日志系统\n- `src/services/error-reporter.ts` - 错误上报（Sentry 集成）\n- `src/services/api-client.ts` - API 统一错误处理\n\n### 使用示例\n\n```typescript\nimport { logger } from '@services/logger';\nimport { apiCall, supabaseCall } from '@services/api-client';\nimport { errorReporter } from '@services/error-reporter';\n\n// 1. 日志记录\nlogger.info('MyModule', 'Operation started', { userId: '123' });\nlogger.error('MyModule', 'Operation failed', error, { context: 'details' });\nlogger.perf('MyModule', 'Query execution', 245); // 性能日志\n\n// 2. API 调用（自动重试）\nconst result = await apiCall('UserService', async () => {\n  const { data } = await supabase.from('users').select();\n  return data;\n}, { maxRetries: 3, timeout: 5000 });\n\nif (result.error) {\n  errorReporter.captureException(\n    new Error(result.error.message),\n    { code: result.error.code }\n  );\n}\n\n// 3. 添加面包屑（用户操作追踪）\nerrorReporter.addBreadcrumb('user-action', 'User clicked button', { buttonId });\n```\n\n### 初始化（应用启动时）\n\n```typescript\n// app/(tabs)/index.tsx\nimport { errorReporter } from '@services/error-reporter';\nimport { setupGlobalErrorHandlers } from '@services/error-reporter';\n\nuseEffect(() => {\n  // 初始化错误上报（可选）\n  errorReporter.init({\n    dsn: 'https://your-sentry-dsn',\n    environment: __DEV__ ? 'development' : 'production',\n    enabled: !__DEV__,\n  });\n\n  // 安装全局错误处理\n  setupGlobalErrorHandlers();\n}, []);\n```\n\n---\n\n## 2️⃣ 安全存储\n\n### 新增文件\n- `src/services/secure-storage.ts` - 加密存储工具\n\n### 使用场景\n\n```typescript\nimport { secureStorage, encryptedAsyncStorage } from '@services/secure-storage';\n\n// 存储敏感数据（Token、密钥）-> Secure Store\nawait secureStorage.set('auth_token', accessToken);\nconst token = await secureStorage.get('auth_token');\nawait secureStorage.remove('auth_token');\n\n// 存储配置和缓存 -> 加密 AsyncStorage\nawait encryptedAsyncStorage.set('user_settings', {\n  theme: 'dark',\n  language: 'zh',\n});\nconst settings = await encryptedAsyncStorage.get('user_settings');\n```\n\n**数据分类建议**：\n- `secureStorage`: Token、API Key、密钥\n- `encryptedAsyncStorage`: 用户配置、缓存数据\n- `AsyncStorage`: 临时数据、非敏感信息\n\n---\n\n## 3️⃣ 输入验证\n\n### 新增文件\n- `src/utils/validators.ts` - 验证和安全转换\n\n### 使用示例\n\n```typescript\nimport { validators, safeCast, guards, validateMultiple } from '@utils/validators';\n\n// 单个字段验证\nconst phoneValidation = validators.chinesePhone(userPhone);\nif (!phoneValidation.valid) {\n  console.error(phoneValidation.error);\n}\n\n// 多字段验证\nconst result = validateMultiple({\n  username: { value: input.username, validator: validators.nonEmptyString('Username') },\n  email: { value: input.email, validator: validators.email(input.email) },\n  age: { value: input.age, validator: validators.numberInRange(input.age, 18, 120, 'Age') },\n});\n\nif (!result.valid) {\n  result.errors.forEach(err => logger.warn('Form', err));\n}\n\n// 安全类型转换\nconst count = safeCast.toNumber(userInput, 0); // 失败返回 0\nconst isAdmin = safeCast.toBoolean(userData.isAdmin, false);\n\n// 清理用户输入\nconst sanitized = validators.sanitize(userInput); // 移除特殊字符\n```\n\n---\n\n## 4️⃣ 认证服务增强\n\n### 新增文件\n- `src/services/auth-enhanced.ts` - Token 刷新、安全认证\n\n### 新增功能\n- 自动 Token 刷新\n- 安全存储 Token\n- Token 过期检测\n\n### 使用示例\n\n```typescript\nimport { authService, tokenManager } from '@services/auth-enhanced';\n\n// 登录\nconst { user, error } = await authService.verifyOtp(phone, otp);\nif (user) {\n  logger.info('Auth', 'User logged in', { userId: user.id });\n}\n\n// 获取有效 Token（自动刷新）\nconst validToken = await tokenManager.getValidAccessToken();\nif (!validToken) {\n  // Token 过期且刷新失败，需要重新登录\n  await authService.logout();\n}\n\n// 登出\nawait authService.logout();\n```\n\n---\n\n## 5️⃣ Zustand Store 优化\n\n### 新增文件\n- `src/stores/store-utils.ts` -
