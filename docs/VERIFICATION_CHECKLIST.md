module`**  \nA: 可能路径别名未配置。检查 `tsconfig.json` 和 `babel.config.js` 中的路径映射。\n\n**Q: Store selector 每次都返回新对象**  \nA: 避免在 selector 中创建新对象。使用预定义的 selector。\n\n**Q: API 调用仍然失败**  \nA: 检查是否所有调用都被 `apiCall()` 包装。某些调用可能被遗漏。\n\n**Q: 日志太多**  \nA: 检查 `logger.debug()` 是否被禁用。生产环境应该自动禁用。\n\n**Q: 性能仍然慢**  \nA: 使用 `PerformanceMark` 找出瓶颈，然后有针对性地优化。\n\n---\n\n## 📞 联系支持\n\n如有问题，请参考：\n1.
