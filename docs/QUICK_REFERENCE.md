read property 'get' of undefined` | Store 初始化前访问 | 等待 `hydrateFromStorage()` 完成 |\n| Token 过期失败 | Token 管理器未初始化 | 启动时调用 `tokenManager.init()` |\n| 验证总是失败 | 输入格式不匹配 | 检查 `validators` 的正则表达式 |\n| Store 频繁重渲染 | 没有使用 Selector | 改为 `useAudioStore(s => s.field)` |\n| 内存泄漏警告 | 忘记清理订阅 | 在 `useEffect` 返回清理函数 |\n| 日志看不到 | 生产环境自动禁用 debug 日志 | 使用 `info/warn/error` |\n\n---\n\n## 文件和文件夹导航\n\n```\n📂 src/\n  ├─
