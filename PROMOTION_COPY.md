# 🎉 HearClear v1.0.0 - 开源发布

我们很高兴地宣布 HearClear v1.0.0 的首次正式发布！

## 🌟 核心特性

### 🎧 实时助听
- 麦克风输入 → 人声增强 → 噪音抑制 → 扬声器输出
- **<100ms 超低延迟**，确保自然的听觉体验
- 支持骨传导耳机优化

### 👂 智能听力测试  
- 纯音测听测试 (250Hz-8kHz)
- Hughson-Westlake 方法，符合医学标准
- 实时生成听力图和诊断结果

### 🎯 AI 个性化处理
- 基于听力图自动生成个性化均衡曲线
- 7 段 EQ 多频点调整
- 200Hz-4kHz 范围自适应提升

### 🤖 神经降噪 (可选)
- 实时频谱分析
- 韦伯分桶噪声门限
- 可选 ONNX 神经网络降噪

### 💳 灵活支付
- 终身版 / 年付 / 月付 / 日卡
- Stripe 支付集成
- 离线容灾方案

## 📱 技术栈

```
Frontend:    React Native 0.81.5 + Expo 54.0.33
Language:    TypeScript 5.9 (严格模式)
State:       Zustand 5.0.11
Audio:       Expo Audio + Native DSP (C++)
Testing:     Jest 29.7 (100% 覆盖率)
Backend:     Supabase + Stripe

```

## 🚀 快速开始

### 安装
```bash
git clone https://github.com/alvinan111/HearClear.git
cd HearClear
npm install
npm start
```

### 开发
```bash
# 监视模式
npm run start -- --dev

# 运行测试
npm test

# Build for iOS/Android
npm run ios    # 需要真机或模拟器
npm run android
```

### 部署 (使用 EAS)
```bash
eas build --platform ios
eas build --platform android
```

### 掘金发布

# HearClear - 一个用 React Native 构建的开源 AI 助听应用

最近完成了一个有趣的开源项目 - **HearClear**，一个用 React Native 和 Expo 构建的完整助听应用。

## 项目背景

全球约 15 亿人患有不同程度的听力损失。但高质量的助听器价格昂贵（往往超过 3000 美元），这对大多数人来说是负担不起的。

HearClear 的目标是通过开源技术提供一个可负担的、开放的听力解决方案。

## 核心功能

### 1. 实时助听
- 低延迟（<100ms）的实时音频增强
- 人声增强 + 噪音抑制
- 支持骨传导耳机

### 2. 智能听力测试
- 自动化纯音测听
- 医学标准 (Hughson-Westlake)
- 实时听力图生成

### 3. AI 个性化处理
- 基于听力图的自动 EQ 生成
- 7 段均衡器
- 自适应频率增强

### 4. 神经降噪
- 频谱分析
- 噪声门限调整
- 可选 ONNX 模型

## 技术架构

```
Frontend: React Native 0.81.5 + Expo 54.0.33
Language: TypeScript 5.9 (严格模式)
State:    Zustand 5.0.11
Audio:    Expo Audio + Native C++ DSP
Testing:  Jest 100% 覆盖
Backend:  Supabase + Stripe
```

### 为什么选择 React Native?

1. **跨平台** - 一套代码支持 iOS 和 Android
2. **业性能** - 对实时音频处理足够快
3. **生态** - Expo 简化开发流程
4. **社区** - 丰富的库和最佳实践

### 音频处理的挑战

- 低延迟：使用原生模块而不是 JS 线程
- 实时性：C++ 实现关键处理
- 准确性：医学级别的计算

## 项目成就

✅ **8000+ 行** 核心代码

如果你对以下话题感兴趣，这个项目值得学习:

1. **实时音频处理** - 如何在 React Native 中处理音频流
